
# Zest

A Video Editor using ImGui with SDL3 and OpenGL as well as FFmpeg for rendering.


![Logo](https://github.com/flosti306/zest/tree/main/Logo/Zest_Logo_Mockup.png)


## Features

- Live previews
- Fullscreen mode
- Cross platform(maybe?)
- Modular window design
- Layering
    - with blend modes and oppacity
- Transformations
- Keying
- Effects
  - Blur
  - Color Grading
- Trimming
