# Zest

Zest is a C++ project designed to composite and cut videos using ImGui with SDL3 and OpenGL for the Frontend as well as FFmpeg.

## Features

- Feature 1: Nothing rn :/
- Feature 2: Nothing rn :/
- Feature 3: Nothing rn :/

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/flosti306/zest.git
    ```
2. Navigate to the project directory:
    ```bash
    cd SDL3VideoEditor
    ```
3. Build the project using your preferred C++ build system (e.g., CMake, Make).
